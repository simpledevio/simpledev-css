simpledev.css

The goal of simpledev.css is to create a mostly classless CSS framework.

The classless part means that most of the CSS code should just work without you having to add classes or IDs to your HTML code. You can just add the CDN links and transform your HTML page. However, we do have a handful of CSS classes that we use for our components, so that's why we call it a mostly classless framework.

We are starting off with working on a handful of components and will add the classless code later. This project is inspired by Water.css, Pico, and Bootstrap.

Pico is the closest framework to this project, but there are a few differences between simpledev.css and Pico. For example, all of the source code is written in plain CSS and there are more classes available.

The navbar uses the navbar class. You can add more links by adding more li elements. You can add an active class to whichever link is currently active.

Non-sticky navbar


    .navbar {
      position: static;
    }

Add this code to your custom CSS file to remove the sticky feature from the navbar.

Navbar Message

Test navbar message.

    <div class="navbar-message">Test navbar message.</div>
    <nav class="navbar">...</nav>

You can add a div with the navbar-message class above your nav element with the navbar class. All it does is add a background color and center the text, but you can customize it in your custom CSS file.

Buttons

Anchor Anchor 

    <a href="" class="button">Anchor</a>
    <button type="button">Button</button>
    <input type="button" value="Input">
    <input type="submit" value="Submit">
    <input type="reset" value="Reset">
    <a href="" class="button-outline">Anchor</a>

You can use the button class on an a element to make it look like a button. The other elements that look like buttons do not need a class. Use the class button-outline to create a button with an outline style.

Square corners


    .button,
    .button-floating,
    .button-outline,
    button,
    input[type="button"],
    input[type="submit"],
    input[type="reset"] {
      border-radius: 0;
    }

Add this code to your custom CSS file if you want square corners instead of rounded corners.

You can add a footer to your site using a footer element with the footer class. We use a footer class since the footer element can be used in different contexts. All it does is add a background color, some padding, and centers the text. You can also add an unordered list with a list of links add use the class footer-links to create a row of links.

Images

You can use images without a class. We apply a max width of 100% to the img element so the image stays inside the container. (Photo by Joshua Aragon on Unsplash.)

Forms

You can use form elements without classes. We set the display property to block for the inputs so each one takes up its own line.

Smooth Scrolling

Smooth scrolling is not enabled by default. If you want to enable smooth scrolling, you can add the following code snippet to your custom CSS file.


    html {
      scroll-behavior: smooth;
    }
